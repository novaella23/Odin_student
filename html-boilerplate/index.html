O elemento strong torna o texto em negrito. Ele também marca semanticamente o texto como importante; isso afeta ferramentas, como leitores de tela, que os usuários com deficiência visual usarão para usar seu site. O tom de voz em alguns leitores de tela mudará para comunicar a importância do texto dentro de um elemento forte. Para definir um elemento forte, envolvemos o conteúdo do texto em uma strong tag.

O elemento torna o texto em itálico. Ele também coloca ênfase semanticamente no exto, o que novamente pode afetar coisas como leitores de tela. Para definir um lemento enfatizado, envolvemos o conteúdo do texto em uma em tag.
